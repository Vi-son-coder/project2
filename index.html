học với github

Phần giới thiệu về github


Github là 1 nền tảng giúp các coder mã hóa và lưu các project khi cần trao đổi hoặc khi làm việc theo team các câu lệnh của git:
1. git init: khởi tạo 1 git repository mới trong thư mục hiện tại
2. git add . : thêm toàn bộ các file trong thư mực vào git foder(nếu muốn thêm từng thư mực thì thay dấu "." bằng tên thư mục)
3. git commit -m "node": dùng để xác nhận sửa đổi "-m" = "--move"
4. git branch -M main: chuyển nhánh từ master sang main
5. git remote add origin + link project: dùng để kết nối và điều khiển với project trên github
6. git push -u origin main: dùng để đẩy dữ liệu từ main lên github(có thể dùng với các nhánh khác khi thay tên nhánh)
7. git branch -c + tên nhánh mới: dùng để copy code từ nhánh main sang 1 nhánh khác và tự tạo ra 1 nhánh khác(-c = --copy)
8. git branch: tra cứu các nhánh
9. git branch -a: xem nhánh cả phần local và remote (-a = --all)
10. git pull origin main: dùng để kéo dữ liệu của người khác về để chỉnh sửa
11. git checkout: dùng để xem mình đang ở nhánh nào
12. git branch -d: dùng để xóa nhánh trên máy(-d = --delete)
13. git branch -d origin + ten nhánh: dùng để xóa nhánh trên github
